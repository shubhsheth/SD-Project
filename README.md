### Installation and Starting (client)

1. `git clone https://github.com/shubhsheth/SD-Project`
2. `cd client`
3. `npm i`
4. `npm start`

### Installation and Starting (server)
1. `git clone https://github.com/shubhsheth/SD-Project`
2. `cd server`
3. `npm i`
4. `nodemon index.js OR node index.js`
